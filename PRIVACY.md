# Privacy Notes

`Agent Pet` 的目标是一个本地优先的桌面监控工具。

## 当前会接触到的数据

应用可能读取或处理这些本地信息：

- 进程名和命令行匹配关键字
- 本地日志路径
- 日志中的最近几行摘要
- localhost / 127.0.0.1 的本地页面入口
- 你手动填写的 Agent 配置

## 当前设计原则

- 优先在本地完成状态判断
- 不把原始日志默认上传到远端
- 让用户能看到和修改每个 Agent 的探测路径
- 不要求用户改源码才能适配自己的环境

## 当前版本的现实边界

这是一个 GitHub alpha 版本，不是已经完成隐私合规审计的商业软件。

在你公开发布前，建议至少确认：

- 是否会把任何调试信息上传到远端
- 是否会记录崩溃日志
- 是否会在 release 版本里接入自动更新
- README 是否已经明确说明读取了哪些本地信息

## 对发布者的建议

- 不要默认采集全文日志
- 如果以后要做错误上报，默认应关闭，并提供明确开关
- 如果以后要上架应用商店，需要补充正式隐私政策和数据用途说明
